# godpaste

A twitter bot that tweets [Digikala's](http://digikala.com) special offers everyday.

## Installation
```bash
go get github.com/erbesharat/godpaste
```

## Usage
1. Change .env folder information with your own API keys.
2. Run code ``` go run dgkala-tb.go ```
